# Social Media Scheduler Tool

MadHub's powerful automation features enable you to manage multiple social media accounts effortlessly. Specifically tailored for social media scheduling, MadHub includes robust functionalities such as **Facebook Group Auto-Posting** and **Facebook Auto-Reply**, making it an essential tool for marketers aiming to enhance their online presence and engagement.

## Feature Overview

### Facebook Group Auto-Posting

With the **Facebook Group Auto-Posting** feature, MadHub allows users to execute automated posting tasks across multiple groups seamlessly. This feature is designed to boost marketing efficiency by streamlining content distribution in various communities. You can set specific parameters for your posts, including content mode (input mode or file mode) and rotation settings, to ensure diverse content reaches your audience without manual input.

**Capabilities:**
- Post content in multiple groups simultaneously.
- Supports content rotation and loop mode for repeated postings.
- Customize the number of posts per group and total post counts.
- Insert images to enrich your posts and attract more engagement.

### Facebook Auto-Reply

The **Facebook Auto-Reply** feature enhances customer engagement further by automatically detecting and replying to unread messages on your Facebook account. This allows you to maintain consistent communication with your audience, ensuring that inquiries are addressed in a timely manner, which is crucial for building trust and rapport with potential customers.

**Capabilities:**
- Automatically responds to messages based on predefined responses.
- Supports both input mode and file mode for versatile reply options.
- Configure operation intervals and duration for optimal engagement frequency.
- Delete messages after sending to keep your inbox organized.

## How to Use These Features

### Setting Up Facebook Group Auto-Posting

1. **Select Groups**: Choose the Facebook groups you want to post in from your dashboard.
2. **Choose Content Mode**: Decide whether to use input mode (manually entering text) or file mode (uploading documents).
3. **Configure Post Settings**:
   - Set the **total post count** and the **number of posts per group**.
   - Enable **content rotation** for varied posts.
   - Optionally, select images to accompany your posts.
4. **Schedule Posts**: Choose the time intervals for your posts to maximize visibility and engagement.

### Setting Up Facebook Auto-Reply

1. **Access Messaging Settings**: Navigate to the auto-reply section in your MadHub dashboard.
2. **Define Auto-Reply Messages**: Input your standard replies or upload a file containing responses.
3. **Configure Parameters**:
   - Set the **operation interval** for how often MadHub checks your inbox.
   - Define the **duration** for auto-replies to run (e.g., during business hours).
4. **Activate Feature**: Once configured, activate the auto-reply and let MadHub handle incoming messages while you focus on other tasks.
